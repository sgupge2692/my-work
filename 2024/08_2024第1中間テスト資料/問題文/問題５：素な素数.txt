### 問題文

2桁以上の整数nは，以下の代入文で1の位を落とすことができる．
```
int n;
n = n / 10;
```
m桁の整数は，m-1回，1の位を落とすことができる．

2桁の素数のうち，1の位を落としても素数であるもの（2桁の素な素数）は，｛23，29，31，37，53，59，71，73，79｝である．
入力された値(max)以下である2桁の素な素数を表示するプログラムを初期ソースコードに示す．

3桁の素数のうち，1の位を1回落としても素数であり，
さらにもう1回落としても素数であるもの（3桁の素な素数）は
｛233，239，293，311，313，317，373，379，
593，599，719，733，739，797｝である．

4桁（1000以上9999以下）の素数のうち，1の位を1回落としても，2回落としても，3回落としても，いずれも素数（4桁の素な素数）であり，入力された値(max)以下であるものをすべて表示するプログラムを作成しなさい．

#### 入力される値
4桁の自然数(max)

#### 期待される出力値
max以下の4桁の素な素数のすべて

#### 初期ソースコード
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
#include <iostream>
using namespace std;

int main() {
    int max;
    cin >> max;
    int num2, num1;
    for (num2 = 10; num2 <= max; num2++) {
        int isPrimeNumber = true;
        for (int i = 2; i <= num2 / 2; i++) {
            if (num2 % i == 0) {
                isPrimeNumber = false;
                break;
            }
        }
        if (isPrimeNumber == true) {
            num1 = num2 / 10;
            if (num1 == 1) {
                isPrimeNumber = false;
            }
            for (int i = 2; i <= num1 / 2; i++) {
                if (num1 % i == 0) {
                    isPrimeNumber = false;
                    break;
                }
            }
        }
        if (isPrimeNumber == true) {
            cout << num2 << endl;
        }
    }
    return 0;
}
```
