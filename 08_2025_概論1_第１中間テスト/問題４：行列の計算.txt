#### __問題文__
***

4×3の行列Xと，3×5の行列Yが標準入力で与えられる．
行列Xと行列Yの行列積を計算し，
得られた4×5の行列の各要素に対して底がeの自然対数（log）を取り，  
その値を小数第3位で四捨五入して出力するプログラムを作成せよ．
ただし，行列積の結果が0以下になる場合は，-1を出力せよ．

なお，自然対数関数logを使うためにインクルードすべき
ライブラリ（ヘッダーファイル）は"cmath"であるので，
自分で初期ソースコードに追加すること．
なお，"cmath"をインクルードすると，
「a = log(b);」でbの自然対数がaに代入される．

#### __入力される値__
続いてYの3行5列の整数  
各要素は半角スペース区切りで与えられる．


#### __期待される出力値__
行列XとYの積の各要素に自然対数（log）を取り，  
小数第3位で四捨五入して表示する．  
行列の要素が0以下になる場合は-1と表示する．  
各行の出力は要素を半角スペースで区切り，行末に改行を入れる．  

***
####__初期ソースコード__
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
#include <iostream>
using namespace std;

int main() {
    int X[4][3], Y[3][5];
    double Ans[4][5];  // log計算結果は実数になるため double を使用

    // 入力
    for (int i = 0; i < 4; i++)
        for (int j = 0; j < 3; j++)
            cin >> X[i][j];

    for (int i = 0; i < 3; i++)
        for (int j = 0; j < 5; j++)
            cin >> Y[i][j];

    /*** ここに回答を記入 ***/

    // 演算した後の行列を出力
    for (int i = 0; i < 4; i++) {
        for (int j = 0; j < 5; j++) {
            // 小数点第3位で四捨五入
            cout << round(Ans[i][j] * 100) / 100 << " ";

        }
        cout << endl;
    }
    cout << endl;

    return 0;
}
```
