###__問題文__
問題６：ユーザが、旅行日数・総予算・興味のあるジャンル番号を入力する。ジャンル番号が一致する観光地から，総予算を上回らない観光地（すなわち，１日当たりの費用×旅行日数が総予算以下である観光地）をすべて表示するプログラムを作成せよ。ただし、1-3以外のジャンル番号を入力した場合は「入力エラー」と出力すること。
観光地の候補は、以下の通りである。
| ジャンル| 1日あたり費用|提案する観光地 |
| ---- | ---- | ---- |
| 1.文化 | 50 | オーストラリア |
| 1.文化 | 20 | 京都 |
| 1.文化 | 10 | 長崎 |
| 2.自然 | 30 | 北海道 |
| 2.自然 | 18 | 鳥取砂丘 |
| 2.自然 | 15 | 阿蘇 |
| 3.歴史 | 70 | ローマ |
| 3.歴史 | 40 | アンコールワット |
| 3.歴史 | 30 | 鎌倉 |


####__入力される値__
3個の自然数(旅行日数、総予算、ジャンル番号)

####__期待される出力値__
観光地（※該当なしの場合は出力なし）

####__初期ソースコード__
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
#include <iostream>
using namespace std;

int main() {
    int travel_days;
    int budget;
    int interest;

    // ユーザーが日数／予算／興味を入力
    cin >> travel_days >> budget >> interest;

        cout << "オーストラリア" << endl;
        cout << "京都" << endl;
        cout << "長崎" << endl;
        cout << "北海道" << endl;
        cout << "鳥取砂丘" << endl;
        cout << "阿蘇" << endl;
        cout << "ローマ" << endl;
        cout << "アンコールワット" << endl;
        cout << "鎌倉" << endl;
        cout << "入力エラー" << endl;

    return 0;
}
```
