### 問題文
以下に示す運転免許の停止・取消ルールに従い、免許保有者の10年間の状態を表示するプログラムを作成しなさい。

1年ごとに違反点数を入力する。違反点数とその累積点数に応じて、以下のいずれかを表示する。
・ゴールドドライバー（3年連続で違反点数が0点）※１
・優良ドライバー（違反点数が0点）
・一般ドライバー（違反点数が1点以上で累積点数が4点以下）
・免停ドライバー（累積点数が5点～9点）※２
・免許取消（累積点数が10点以上）※３

※１：ゴールドになったら累積点数は0点に戻る。
※２：免停になった場合は累積点数は0点に戻る。
　　　翌年の状態は“免停中”
※３：免許取消でプログラム終了

#### 入力される値
0以上の整数

#### 期待される出力値
状態を示す文字列

#### 初期ソースコード
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
#include <iostream>
using namespace std;

int main() {
    int points;             //違反点数
    int totalPoints = 0;    //これまでの違反点数の累積点数
    int noViolations = 0;   //無違反年数（違反0点の年数）

    for (int year = 1; year <= 10; year++) {
        cout << year << "年目　違反点数 ";
        cin >> points;
        totalPoints += points;
        
        cout << "_______状態：ゴールドドライバー";
        totalPoints = 0;

        cout << "_______状態：優良ドライバー";


        cout << "免許取消";
        return 0;

        totalPoints = 0;
        cout << "_______状態：免停ドライバー" << "，累積点数：" << totalPoints << endl;
        if (year == 10) {
            return 0;
        }
        year++;
        cout << year << "年目　状態：免停中________";


        cout << "_______状態：一般ドライバー";

        cout << "，累積点数：" << totalPoints << endl;
    }
    return 0;
}
```
