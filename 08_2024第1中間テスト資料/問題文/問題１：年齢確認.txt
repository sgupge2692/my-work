###__問題文__
問題１：年齢を入力したら、成年かどうか、飲酒・喫煙が可能かどうかを改行して出力するプログラムを作成せよ。

18歳未満なら「未成年」「飲酒・喫煙不可」
18歳以上20歳未満なら「成人」「飲酒・喫煙不可」
20歳以上なら「成人」「飲酒・喫煙可」
と出力せよ。

####__入力される値__
1つの整数値（年齢）

####__期待される出力値__
1つの文字列（未成年 or 成年）
1つの文字列（飲酒・喫煙可 or 飲酒・喫煙不可）

***
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
####__初期ソースコード__
#include <iostream>
using namespace std;
int main() {
        cout << "未成年" << endl << "飲酒・喫煙不可" << endl;
        cout << "成人" << endl << "飲酒・喫煙不可" << endl;
        cout << "成人" << endl << "飲酒・喫煙可" << endl;
}
```
