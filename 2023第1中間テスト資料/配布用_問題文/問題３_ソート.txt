###__問題文__
***
問題３：10個の自然数が入力されます．
以下のように並べ替えて（ソートして）出力するプログラムを作成しなさい．

・奇数と偶数では奇数を前（先）とする．

・奇数同士では，昇順とする．

・偶数同士では，降順とする．

####__入力される値__
***
10個の自然数

####__期待される出力値__
***
前に奇数を昇順に，後に偶数を降順に並べ替えた10個の自然数

####__制約__
***
同値が入力される場合がある．

***
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
####__初期ソースコード__
#include <iostream>
#define N 10
using namespace std;

int main() {
    int nums[N];

    for (int i = 0; i < N; i++) {
        cin >> nums[i];
    }

    //奇数を昇順にソートして出力
    cout << nums[i] << ",";

    //偶数を降順にソートして出力
    cout << nums[i] << ",";

}


```
