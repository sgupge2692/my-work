###__問題文__
***
問題５：あなたは，4桁の番号が書かれた宝くじを10枚購入しました．
10枚分の宝くじの番号は以下の通りです．
int lot[10] = { 7851, 2453, 7906, 2919, 3811, 2438, 4453, 2335, 4456, 2867 };
当選番号(int win_num)をユーザ入力します．
宝くじの当選枚数と，当選金額の合計を出すプログラムを書きなさい．

当選の判定と当選金額は，以下の通りである．
-----------------------------
1等：全4桁が同一：5,000,000円
2等：下3桁が同一：100,000円
3等：下2桁が同一：5,000円
4等：下1桁が同一：300円
-----------------------------


####__入力される値__
***
4桁の自然数（当選番号)

####__期待される出力値__
***
2つの整数値(宝くじの当選枚数，当選金額の合計)

####__制約__
***
入力される自然数は，1000以上，9999以下とする．

***
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
####__初期ソースコード__
#include <iostream>
using namespace std;

int main() {
    int lot[10];
    int prize1 = 0, prize2 = 0, prize3 = 0, prize4 = 0;
    int prize_total = 0;

    cin >> win_num; //当選番号

    /*ここに回答を記入*/

    int prize_total = prize1 * 5000000 + prize2 * 100000 + prize3 * 5000 + prize4 * 300;
    cout << "当選枚数：" << prize1 + prize2 + prize3 + prize4 << endl;
    cout << "当選金額：" << prize_total << endl;
}
```
