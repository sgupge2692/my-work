###__問題文__
***
問題１：三角形の3つの頂点の角度(degree)を
inr angle1, angle2, angle3に自然数で入力します．

入力値から，鋭角三角形，鈍角三角形，直角三角形の
どれかを判定するプログラムを作成しなさい．
ただし，三角形にならない場合
（3つの頂点の角度の和が180ではない場合）の処理も行うこと．

####__入力される値__
***
3つの自然数（三角形の頂点の角度）

####__期待される出力値__
***
指定されたいずれかの文字列（三角形の種類の判定結果）

####__制約__
***
自然数以外（0や負の数，分数など）は入力されないものとする．

***
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
####__初期ソースコード__
#include <iostream>
#include <string>
using namespace std;

int main() {

    int angle1, angle2, angle3;
    cin >> angle1 >> angle2 >> angle3;  //三角形の角度を入力


    //三角形の判定結果の表示
    cout << "三角形ではない" << endl;    //三角形にならない場合
    cout << "鋭角三角形" << endl;    //鋭角三角形の場合
    cout << "鈍角三角形" << endl;    //鈍角三角形の場合
    cout << "直角三角形" << endl;    //直角三角形の場合
        
}


```
