#### __問題文__
***
整数 $A,B$ が与えられます。
$A$ と $B$ の最小公倍数を求めてください。

#### __入力される値__
***
```
A B
```
- $1$ 行目に整数 $A,B$ が半角スペース区切りで与えられます。

#### __期待される出力値__
***
一行で $A$ と $B$ の最大公約数を出力してください。	

#### __制約__
***
- 入力はすべて整数である。
- $1 \leq B \leq A \leq 100$


***
####__初期ソースコード__
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```
#include <iostream>
using namespace std;

int main() {
	int a, b, ans;
	cin >> a >> b;
	if (a > b) {
		ans = a;
	}
	else {
		ans = b;
	}

//ここに回答を入力（「while (true)」で無限ループ）

	cout << ans << endl;
	return 0;
}
```
