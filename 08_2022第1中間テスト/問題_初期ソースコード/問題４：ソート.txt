###__問題文__
***
問題４：ソート

5つの実数が入力されます．
昇順（小さい順）に並べ替えて出力するプログラムを作成しなさい．

####__入力される値__
***
5つの実数

####__期待される出力値__
***
昇順に並べ替えられた5つの実数

####__制約__
***
正の数か負の数かは問わない．
同値が入力される場合がある．
double型で丸め誤差やオーバーフロー等が発生しない値とする．

####__初期ソースコード__
***
（コピー＆ペーストしてもよい，cout文は変更しないこと）
```

#include <iostream>
using namespace std;

int main() {
	double data[5];
	for (int i = 0; i < 5; i++) {
		cin >> data[i];
	}

	/*回答部分*/

	for (int i = 0; i < 5; i++) {
		cout << data[i] << endl;
	}
}
```
